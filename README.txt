# Abaxus — Static Site (Drag & Drop)

This folder contains a single‑page website you can deploy by dragging it into Vercel.

## How to publish (no code needed)
1) Go to https://vercel.com → Sign up / Log in
2) Click **Add New… → Project → Import** and choose **Drag & Drop**.
3) Drag this entire folder into the upload area and click **Deploy**.
4) You’ll get a live URL. To add your domain:
   - Project → Settings → **Domains** → Add your domain (e.g., abaxus.com)
   - Follow the on‑screen DNS instructions.

## Edit contact details
Open `index.html` and edit these spots:
- Header & CTA `mailto:hello@abaxus.com`
- Phone number `(555) 123‑4567`
- Company name `Abaxus`

Save and re‑upload to Vercel (or connect a Git repo for auto‑updates).
